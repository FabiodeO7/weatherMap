# 🚀 INÍCIO RÁPIDO - WeatherMap

## ⚡ 3 Passos para Começar

### 1️⃣ Abrir a Aplicação
No navegador, acesse:
```
http://localhost:8000
```

*A aplicação já está rodando em servidor local*

### 2️⃣ Buscar uma Cidade
- Digite o nome em português ou inglês
- Pressione Enter ou clique em "Buscar"
- Exemplos: São Paulo, Paris, Tóquio, New York

### 3️⃣ Visualizar Dados
Você verá:
- 📸 Imagem da cidade
- 🌡️ Temperatura atual
- 💧 Umidade, vento, pressão, visibilidade
- 📅 Previsão de 5 dias

## 🐛 Problemas Comuns

**P**: A página não carrega?
**R**: Verifique se o servidor está rodando (terminal com "python -m http.server 8000")

**P**: Imagem não aparece?
**R**: Verifique sua conexão de internet

**P**: Temperatura não mostra?
**R**: Tente recarregar a página (F5)
